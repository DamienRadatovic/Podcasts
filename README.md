# Podcasts Website

Website to listen and promote podcasts. All data ara static and fictive.

The purpose of this project is to train me on the React framework.

## Installation

Install with npm

```bash
  npm install
  npm run dev
```

## Tech Stack

**Client:** React, Css, Typescript, WaveSurfer, SWR
